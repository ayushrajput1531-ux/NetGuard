# NetGuard v1.0

NetGuard is a Professional Network Diagnostic Tool built with Python.

## Features

- Internet Check
- DNS Lookup
- Public IP
- Private IP
- Ping Test
- Port Scanner
- WiFi Information
- Network Adapter Information
- System Information
- Internet Speed Test
- Network Health Score
- PDF Report
- Scan History

## Technologies Used

- Python
- Tkinter
- ReportLab
- Speedtest CLI
- PyInstaller
